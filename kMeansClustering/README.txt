This project try to achive a quality measure for a k number of clusters from a set of points.

In order to run the project required 3 arguments to be given
 - exe
 - full path to input file
 - full path to output file
 
 Note: 
 tests can done by upping the value of define FIND_MAX_RUN_PER_POINT inside Cuda filter inside cuda.cu file.
 this would improve preformance, beware those changes can crush the system depending on GPU and the data sets given.
